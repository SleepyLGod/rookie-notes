# Distributed Systems

这一节放分布式系统和经典 Google 系统论文笔记。

建议阅读顺序：

1. [CAP theorem](cap-theorem.md)：先明确一致性、可用性和分区容忍性的边界。
2. [Google File System](gfs.md) 与 [Google MapReduce](mapreduce.md)：理解大规模数据处理的存储和计算基础。
3. [Google Bigtable](bigtable.md)：最后看面向结构化数据的分布式存储模型。
