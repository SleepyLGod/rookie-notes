# Systems

This section organizes notes on distributed systems, storage engines, classic Google system papers, and LevelDB/RocksDB/TiKV.

Suggested reading order:

1. Start with [Distributed Systems](distributed-systems/README.md), including CAP, GFS, MapReduce, and Bigtable.
2. Then read [Storage Engines](storage-engines/README.md), including LSM-tree, SSTable, filters, and mainstream KV implementations.
3. Finally, revisit storage-layer design through the concrete implementation notes on LevelDB, RocksDB, and TiKV.

Content status: this group contains original summaries, course/paper reading notes, and some rewritten material from external sources. For version-specific behavior, use the current official documentation or source code of the corresponding project as the source of truth.
