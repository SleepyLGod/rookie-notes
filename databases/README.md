# Databases

This section mainly contains database course notes, implementation notes, and GPU database materials.

Suggested reading order:

1. [CMU 15-445](cmu-15-445/README.md): database systems fundamentals and BusTub project implementation, including buffer pool, B+ tree, query execution, and concurrency control.
2. [CMU 15-721](cmu-15-721/README.md): advanced database systems, including in-memory DBMS, MVCC, OLTP indexes, and related topics.
3. [GPU Databases](gpu-databases/README.md): entry materials on the intersection of GPU and database systems.

Content status: the 15-445 section contains many project implementation details and may be tightly coupled to a specific course year. The 15-721 section is closer to lecture summaries. When reviewing later, keep the course year and project version context.
