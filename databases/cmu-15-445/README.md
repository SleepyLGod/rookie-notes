# CMU 15-445

These are notes on CMU 15-445's BusTub projects, organized by implementation module.

Suggested reading order:

1. [Buffer Pool](buffer-pool/README.md): pages, frames, LRU replacer, and buffer pool manager.
2. [B+ Tree Index](b-plus-tree-index/README.md): page layout, insertion, deletion, iterator, and concurrent index.
3. [Query Execution](query-execution.md): catalog, executor, scan, insert/update/delete, and join.
4. [Concurrency Control](concurrency-control.md): lock manager, isolation levels, and transaction states.

Content status: these notes depend heavily on the course project version. For interface names, test commands, or lock-protocol details, use the BusTub handout and source code from the corresponding year as the source of truth.
